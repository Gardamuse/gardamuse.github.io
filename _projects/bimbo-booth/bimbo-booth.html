Bimbofication Booth

Contains sexual content with focus on Bimbofication. This includes Breast Expansion, TF/TG and TF/Ment.

Hotkeys

R - Reset, go to first frame
P - Play / Pause
B - Back, go to previous frame
N - Next, go to next frame

A - Select previous character
D - Select next character

F - Fullscreen toggle
C - Character select toggle

Some YouTube-like hotkeys also available, not listed.

Improvements

While the app has been working well for me, there are plenty of devices and browser I can't test extensively on my own. So let me know if you encounter any bugs or if something is performing poorly, but be sure to tell me what device/browser you are using!
